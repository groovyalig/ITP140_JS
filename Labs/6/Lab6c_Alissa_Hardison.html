This page will create a username and email for a website.

The username will be created using the first initial of the user's first name appended to first four characters of the last name.

The email will be created using the first initial of the user's first name appended to the last name, '@', school name and the domain '.edu'

For example:

  • First name = Cesar
  • Last name = Barbieri
  • School name = ODU

Username would be: Cbarb

Email would be: cbarbieri@odu.edu
